介紹/Introduction:

此乃開發中的"熱血少年"同人/二次創作3D遊戲系統的網頁測試版. 暫時只支援Google Chrome/Firefox瀏覽器(理論上任何平台版本皆可, 包括手機).
3D模型可隨意轉動. 頁面右上角按鈕可將viewport放大/縮小. 左下/上角有按鈕可以選擇任何MP3/AAC/WAV音樂檔, 隨後3D角色會自動因應節拍而起舞.

This is a technical preview of the upcoming 3D game system based on derivative works of characters from "PassionTeens" online comics. For now, it works on Google Chrome/Firefox web browser only (including mobile phones).
3D model can be freely rotated using your mouse. You can use the button on the upper right to change the viewport size. You can use the button on the lower/upper left to choose any MP3/AAC/WAV file, and the 3D character will dance to the beats automatically.
