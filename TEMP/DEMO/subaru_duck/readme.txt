Introduction:

  Use the "File input" button on the lower/upper left to choose any MP3/AAC/WAV file (or simply drop the music file to the page), and the "Subaru and Duck" duo will dance to the beats automatically.

  This music visualizer is powered by "System Animator" online version, which now works on web browsers instead of being just a desktop gadget.

  For more information about System Animator, please visit the following page.
  http://www.animetheme.com/sidebar/
