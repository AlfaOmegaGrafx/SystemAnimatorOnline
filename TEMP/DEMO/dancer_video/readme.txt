Introduction:

  Use the "File input" button on the lower/upper left to choose any MP3/AAC/WAV file (or simply drop the music file to the page), and the girl will dance to the beats automatically.

  This music visualizer is powered by the PREVIEW version of "System Animator 11", which now works (partially) on web browsers (Google Chrome/Firefox only, for now) instead of being just a desktop gadget.

  For more information about System Animator, please visit the following page.
  http://www.animetheme.com/sidebar/
